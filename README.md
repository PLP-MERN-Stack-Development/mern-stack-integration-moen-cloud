# BlogSphere

BlogSphere is a full-stack blogging platform built with the MERN stack (MongoDB, Express, React, Node.js) and styled using Tailwind CSS. It supports user authentication, role-based access control, post creation, and category management â€” all wrapped in a clean, responsive UI.

---

##  Features

- JWT-based authentication (Login/Register)
-  Role-based access (Admin vs User)
-  Create, read, and list blog posts
-  Category selection for posts
-  Protected routes using React Router
-  Tailwind CSS for modern UI
-  Vite for fast frontend development

--- Future Enhancements
BlogSphere is still evolving. Planned improvements include:

 Add blog creation UI: A rich editor for users to write and publish their own blog posts, with markdown or WYSIWYG support.

 View existing blogs: A public-facing blog feed where users can browse, read, and engage with published posts.

Category manager: Admin interface to create, edit, and delete blog categories.

 User profiles: Each user will have a profile page showing their published posts and activity.

 Search and filter: Ability to search blogs by title, category, or author.

 Dark mode toggle: Theme switcher for better accessibility and user preference.

 Comments and likes: Social features to engage with blog content.
